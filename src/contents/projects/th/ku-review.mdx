## ภาพรวม

KU Review เป็นแพลตฟอร์มรีวิวรายวิชาที่ครอบคลุมสำหรับนิสิตมหาวิทยาลัยเกษตรศาสตร์ แพลตฟอร์มนี้ช่วยให้นิสิตแชร์ประสบการณ์การเรียน ให้คะแนนรายวิชา และช่วยเพื่อนนิสิตตัดสินใจเลือกรายวิชาได้อย่างเหมาะสม

## คุณสมบัติ

- ค้นหาและเรียกดูรายวิชาจากคณะต่างๆ
- ให้คะแนนรายวิชาในหลายด้าน (เนื้อหา ความยาก คุณภาพการสอน)
- เขียนรีวิวและแชร์ประสบการณ์โดยละเอียด
- ดูคะแนนรวมและรายวิชายอดนิยม
- ค้นหาตามภาคเรียนและอาจารย์
- ออกแบบที่เข้ากันได้กับอุปกรณ์เคลื่อนที่

## เทคโนโลยี

**Frontend**: สร้างด้วย SvelteKit เพื่อให้มีประสบการณ์การพัฒนาที่ดีขึ้น TypeScript ช่วยให้มั่นใจในความปลอดภัยของประเภทข้อมูล ส่วน TailwindCSS ให้สไตล์ที่สอดคล้องและตอบสนองต่อหน้าจอต่างๆ

**Backend**: API ที่สร้างด้วย Node.js พร้อมฐานข้อมูล PostgreSQL สำหรับเก็บข้อมูลถาวร ได้เพิ่มการเพิ่มประสิทธิภาพแบบสอบถามขั้นสูง

**Deployment**: ใช้ Vercel สำหรับการปรับใช้ด้วยการปรับใช้อัตโนมัติจาก GitHub และการแคชขอบเขตสำหรับประสิทธิภาพที่เหมาะสม

## ความท้าทายและวิธีแก้ไข

### ความท้าทาย: การจัดการชุดข้อมูลรีวิวขนาดใหญ่
การแสดงรีวิวหลายพันรายอย่างมีประสิทธิภาพต้องใช้กลยุทธ์การแบ่งหน้า การโหลดแบบเกียจคร่าน และการจัดทำดัชนีฐานข้อมูล

**วิธีแก้ไข**: ใช้การแบ่งหน้าตามเคอร์เซอร์และเพิ่มดัชนีในคอลัมน์ที่ค้นหาบ่อย ลดเวลาการสืบค้นลง 60%

### ความท้าทาย: การรวมคะแนนที่แม่นยำ
การรักษาความสอดคล้องของการคำนวณคะแนนเมื่อมีรีวิวใหม่ต้องจัดการธุรกรรมฐานข้อมูลอย่างระมัดระวัง

**วิธีแก้ไข**: ใช้คอลัมน์ที่คำนวณได้และ trigger ของ PostgreSQL เพื่อคำนวณคะแนนใหม่อัตโนมัติ

## ผลลัพธ์

แพลตฟอร์มได้รับความนิยมจากนิสิต 2,500+ คนในภาคเรียนแรก พร้อมรีวิว 5,000+ รายการ กลายเป็นทรัพยากรหลักสำหรับการเลือกรายวิชา
